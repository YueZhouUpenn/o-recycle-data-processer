# Web界面使用指南

## 功能特性

### ✨ 主要功能
- 📊 **实时数据展示** - 直接查看SQLite数据库中的总台账数据
- 🔍 **智能搜索** - 支持序列号、物料名称、销售员、客户等多字段搜索
- 📈 **统计面板** - 实时显示总出库数、已回收数、未回收数、回收率
- 📥 **Excel导出** - 一键导出当前数据为Excel文件
- 📱 **响应式设计** - 支持电脑、平板、手机等多种设备
- 🎨 **现代化UI** - 美观的渐变色设计，流畅的交互体验

### 🎯 页面特点
- **渐变背景** - 紫色系渐变，专业美观
- **卡片式布局** - 清晰的信息层次
- **状态徽章** - 不同颜色标识不同回收状态
- **分页浏览** - 支持大数据量的高效浏览
- **防抖搜索** - 输入停止后自动搜索，提升性能

## 启动方式

### 方式一：通过命令行菜单启动（推荐）

```bash
# 1. 编译项目
mvn clean compile

# 2. 运行主程序
mvn exec:java -Dexec.mainClass="com.company.recycle.Main"

# 3. 在菜单中选择 "6. 启动Web界面"
```

### 方式二：直接启动Web服务

```bash
# 编译并启动Web服务
mvn spring-boot:run -Dspring-boot.run.mainClass="com.company.recycle.web.WebApplication"
```

### 方式三：使用IntelliJ IDEA

1. 打开项目
2. 找到 `WebApplication.java` 文件
3. 右键选择 `Run 'WebApplication.main()'`

### 方式四：打包后运行

```bash
# 1. 打包
mvn clean package

# 2. 运行JAR
java -cp target/o-recycle-data-processer-1.0.0.jar com.company.recycle.web.WebApplication
```

## 使用说明

### 📊 查看统计信息

页面顶部显示四个统计卡片：
- **总出库数** - 蓝色卡片，显示所有出库记录数
- **已回收数** - 绿色卡片，显示已回收的数量
- **未回收数** - 橙色卡片，显示待回收数量
- **回收率** - 红色卡片，显示回收完成比例

### 🔍 搜索数据

1. 在搜索框中输入关键词
2. 支持搜索以下字段：
   - 序列号
   - 物料名称
   - 销售员
   - 客户
   - 最终客户
3. 输入后自动触发搜索（0.5秒延迟）

### 📄 浏览数据

- 表格显示完整的总台账信息
- 每页显示50条记录
- 回收状态用不同颜色的徽章标识：
  - 🟡 **未回收** - 黄色
  - 🟢 **现场回收** - 绿色
  - 🔵 **统一回收** - 蓝色
  - 🟣 **退货** - 紫色
  - 🔴 **问题序列号** - 红色

### 📥 导出Excel

1. 点击右上角 "📥 导出Excel" 按钮
2. 系统自动生成Excel文件
3. 浏览器自动下载
4. 文件名格式：`销售出库总台账_YYYY-MM-DD.xlsx`

### 🔄 刷新数据

- 点击 "🔄 刷新" 按钮重新加载数据
- 数据会自动从数据库重新读取

### 📃 分页导航

页面底部提供完整的分页控件：
- **首页** - 跳转到第一页
- **上一页** - 上一页
- **页码按钮** - 直接跳转到指定页
- **下一页** - 下一页
- **末页** - 跳转到最后一页

## 常见问题

### Q: 页面无法访问？
**A:** 检查以下几点：
1. 确认服务已启动（查看控制台日志）
2. 确认端口8080未被占用
3. 尝试访问 `http://127.0.0.1:8080`

### Q: 数据显示为空？
**A:** 可能原因：
1. 数据库为空 - 先通过命令行菜单导入数据
2. 数据库文件路径错误 - 检查 `data/recycle.db` 是否存在

### Q: 导出Excel失败？
**A:** 检查：
1. 浏览器是否允许下载文件
2. `output` 目录是否有写入权限
3. 查看控制台日志了解详细错误

### Q: 修改端口号？
**A:** 编辑 `src/main/resources/application.properties`：
```properties
server.port=9090  # 改为你需要的端口
```

### Q: 如何停止服务？
**A:** 在启动Web服务的终端窗口按 `Ctrl + C`
